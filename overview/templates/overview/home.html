歡迎來到我們的網站


大數據時代來臨，如何從眾多資料中抽取出有意義的訊息，分析市場、探勘輿情，對許多政府單位、企業而言是相當重要的應用。 本系統利用中文剖析系統， 保留了上次報告中近期PPT特定版面的新聞語料，新增了從新聞中分析近期熱門的總統大選，從每句的結構樹中找出字詞之間的次數關係， 將資料透過不同圖表進行可視化。

我們的網站旨在提供以下幾項功能：

  • Popular Keywords：您可以在我們的網站上了解目前最多人提起的關鍵詞與其次數。
  • Popular People：您可以在我們的網站上查看目前哪個人物最常被提起與其次數。
  • User Keyword：您可以在我們的網站上可以針對你輸入的個別關鍵詞進行熱門程度分析。
  • User Keyword Association：您可以在我們的網站上對你想要了解的議題進行全文檢索，找出有哪些詞與你的關鍵詞一起出現?
  • User Keyword Sentiment：您可以在我們的網站上可以了解媒體對該關鍵詞的情緒程度。
  • Top Rank People：您可以在我們的網站上了解目前賴清德、侯友宜、柯文哲的人氣聲量大PK
  • Top Person Yesterday：您可以在我們的網站上了解昨日哪個人物被報導最多次?
  • Sentiment Analysis：BERT模型。
